The PhoneGap Developer App

Creating mobile apps is now even easier

Develop on your local machine then push your changes straight to the PhoneGap Developer App.
No need to re-sign, reinstall, or relaunch your app to test your code.

First, install the CLI

You will need to install the Phonegap CLI module using npm. An in depth guide on how to do it is located here.

Next, download the mobile app

Get the PhoneGap Developer App for the platform you are developing on.

Now, pair the CLI to the app

Go to your PhoneGap project within terminal, then start the server. You will see an IP address to input on the start screen of the PhoneGap Developer App on your mobile device.

Feel the Power!

Start making changes to your assets on your machine and see the new changes appear instantly!
